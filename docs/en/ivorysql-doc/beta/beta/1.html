1. Versions Inroduction

[Release Date: June 28, 2023]

IvorySQL 3.0 is based on PostgreSQL 16.0 includes various fixes from PostgreSQL 16.0, For more detailed updates and bug fixes in PostgreSQL 16.0, see the official PostgreSQL 16.0 Release Notes .

3.3. Compatible with PL/SQL

  • Addresses an issue with PL/SQL creation functions/stored procedures 问题细节

  • Creating a function or procedure in SQL parser supports nested subprocedures 问题细节

  • Nested child processes and functions IS/As do not need to be declared 问题细节

  • When parsing, nested child processes retype are obtained from rettype 问题细节
