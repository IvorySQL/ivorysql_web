%ROWTYPEгҖҒ%TYPE

1. зӣ®зҡ„

IvorySQLжҸҗдҫӣдәҶе…је®№Oracleзҡ„plsqlж•°жҚ®зұ»еһӢеҠҹиғҪпјҢеҢ…жӢ¬%TYPEгҖҒ%ROWTYPEгҖӮ

2. е®һзҺ°иҜҙжҳҺ

2.1. еј•з”Ёзұ»еһӢеҸ‘з”ҹеҸҳеҢ–ж—¶пјҢеҰӮдҪ•зЎ®дҝқ%TYPEеЈ°жҳҺзҡ„еҸҳйҮҸд№ҹйҡҸд№Ӣзӣёеә”ж”№еҸҳ

иҝҷжҳҜдёҖдёӘиў«еҠЁзҡ„иҝҮзЁӢгҖӮ еҪ“еүҚзҡ„е®һзҺ°ж–№жЎҲжҳҜи®°еҪ•еҮҪж•°пјҲеӯҳеӮЁиҝҮзЁӢпјүдёҺ tablename.columname зҡ„дҫқиө–е…ізі»гҖӮеҪ“еј•з”Ёзұ»еһӢеҸ‘з”ҹеҸҳеҢ–ж—¶пјҢж №жҚ®дҫқиө–е…ізі»пјҢи®©еҮҪж•°пјҲеӯҳеӮЁиҝҮзЁӢпјүзј“еӯҳеӨұж•ҲгҖӮиҝҷж ·дёҖжқҘпјҢеҮҪж•°иў«и°ғз”Ёж—¶е°ұдјҡжү§иЎҢејәеҲ¶зј–иҜ‘гҖӮд»ҺиҖҢзЎ®дҝқеҮҪж•°иҺ·еҸ–жңҖж–°зҡ„еҸҳйҮҸзұ»еһӢгҖӮ

еңЁзі»з»ҹиЎЁ pg_proc дёӯж·»еҠ дёҖдёӘеӯ—ж®ө prostatus з”ЁжқҘиЎЁзӨәеҮҪж•°пјҲеӯҳеӮЁиҝҮзЁӢпјүзҠ¶жҖҒпјҢжңү3з§ҚзҠ¶жҖҒ: validate(v)пјҢinvalidate(i)пјҢN/A(n)гҖӮеҮҪж•°зј–иҜ‘жҲҗеҠҹеҗҺпјҢиҝҷдёӘзҠ¶жҖҒдјҡиў«и®ҫзҪ®дёәvalidгҖӮ

еңЁи§ЈжһҗеҮҪж•°еҶ…е®№ж—¶пјҢplisql_parse_cwordtypeпјҢplisql_parse_wordrowtypeпјҢplisql_parse_cwordrowtype зӯүеҮҪж•°иҜҶеҲ«еҮә%TYPEпјҢ %ROWTYPEеј•з”Ёзҡ„еҜ№иұЎпјҢе№¶и®°еҪ•еңЁ plisql_referenced_object й“ҫиЎЁдёӯпјҢжңҖеҗҺж·»еҠ еҲ° pg_depend зі»з»ҹиЎЁдёӯгҖӮ

еўһеҠ дёҖз§Қж–°зҡ„дҫқиө–зұ»еһӢ DEPENDENCY_TYPE = 't'пјҢиЎЁзӨә%TYPEжҲ–%ROWTYPEдҫқиө–гҖӮеңЁж·»еҠ еҜ№иұЎеј•з”Ёе…ізі»еҲ°pg_dependзі»з»ҹиЎЁж—¶пјҢи®ҫзҪ®дҫқиө–зұ»еһӢдёә 't'гҖӮ

еҜ№иЎЁиҝӣиЎҢж“ҚдҪңж—¶пјҲдҝ®ж”№иЎЁзұ»еһӢгҖҒеҲ йҷӨиЎЁзӯүпјүпјҢжҹҘзңӢзі»з»ҹиЎЁpg_dependпјҢеҰӮжһңеӯҳеңЁдҫқиө–зұ»еһӢdeptype=вҖҷtвҖҷпјҢе№¶дё”дҫқиө–еҜ№иұЎжҳҜеҮҪж•°пјҢеҲҷи°ғз”ЁеҮҪж•° plisql_free_function еҲ йҷӨеҮҪж•°зј“еӯҳпјҢе№¶дҝ®ж”№ pg_proc зі»з»ҹиЎЁдёӯзҡ„еҮҪж•°зҠ¶жҖҒprostatus дёә N/A(n)гҖӮ

2.2. %TYPEеЈ°жҳҺзҡ„еҸҳйҮҸз»§жүҝеј•з”ЁеҸҳйҮҸзҡ„зәҰжқҹ

еңЁ PLiSQL_type з»“жһ„дҪ“дёӯж·»еҠ жҲҗе‘ҳ bool notnull;

/*
 * Postgres data type
 */
typedef struct PLiSQL_type
{
	bool		notnull;		/* the type is built by variable%type,
								 * isnull or notnull of the variable */

еңЁиҙҹиҙЈи§Јжһҗ%TYPEзұ»еһӢеҮҪж•°зҡ„plisql_parse_wordtypeжҲ–plisql_parse_cwordtypeеҮҪж•°дёӯпјҢеҲӨж–ӯеҰӮжһңеј•з”ЁеҸҳйҮҸзұ»еһӢжҢҮе®ҡдәҶNOT NULLзәҰжқҹпјҢдёәиҝ”еӣһзҡ„datatypeзҡ„жҲҗе‘ҳ bool notnull еұһжҖ§иөӢеҖјдёәtrueгҖӮ еңЁpl_gram.y ж–Үд»¶зҡ„ decl_statement иҜӯжі•дёӯпјҢж №жҚ®PLiSQL_type зҡ„жҲҗе‘ҳ bool notnull дёәеҸҳйҮҸPLiSQL_variable *varзҡ„notnullеұһжҖ§иөӢеҖјгҖӮиҝҷж ·е°ұз»§жүҝдәҶеј•з”ЁеҸҳйҮҸзҡ„зәҰжқҹгҖӮ

2.3. иЎЁеҗҚ%ROWTYPEжҲ–и§ҶеӣҫеҗҚ%ROWTYPEдҪңдёәеҮҪж•°жҲ–еӯҳеӮЁиҝҮзЁӢзҡ„еҸӮж•°зұ»еһӢе’ҢеҮҪж•°иҝ”еӣһеҖјзұ»еһӢ

еңЁ ora_gram.y дёӯдёә func_type ж·»еҠ %ROWTYPEж”ҜжҢҒгҖӮ
