зј–иҫ‘жӯӨйЎөйқў

еҠҹиғҪжҰӮиҝ°

IvorySQLжҸҗдҫӣдәҶе…је®№Oracleзҡ„outеҸӮж•°еҠҹиғҪпјҢеҢ…жӢ¬еҗ«жңүoutеҸӮж•°зҡ„еҮҪж•°дёҺиҝҮзЁӢгҖҒеҢҝеҗҚеқ—ж”ҜжҢҒoutеҸӮж•°гҖҒlibpqж”ҜжҢҒoutеҸӮж•°гҖӮ

1. е®һзҺ°еҺҹзҗҶ

1.1. еҗ«жңүoutеҸӮж•°зҡ„еҮҪж•°

еҜ№дәҺpliSQLеҮҪж•°пјҢеңЁеҲӣе»әеҮҪж•°ж—¶пјҢзі»з»ҹиЎЁpg_procдёӯеӯҳеӮЁеҮҪж•°еҸӮж•°дёәе…ЁйғЁеҸӮж•°дёӘж•°пјҲеҢ…жӢ¬OUTеҸӮж•°дёӘж•°пјүеҸҠеҜ№еә”зҡ„еҸӮж•°ж•°жҚ®зұ»еһӢгҖӮ

еңЁеӨ„зҗҶеҮҪж•°еҸӮж•°зҡ„interpret_function_parameter_list()еҮҪж•°дёӯпјҢж №жҚ®еҸӮж•°жЁЎејҸпјҢеҲӨж–ӯеҰӮжһңжҳҜIN OUTпјҢеҲҷеҸӮж•°дёҚиғҪжңүй»ҳи®ӨеҖјгҖӮ

еңЁmake_return_stmtеҮҪж•°дёӯпјҢеҺ»жҺүеҰӮжһңеҸ‘зҺ°жңүoutеҸӮж•°жҠҘй”ҷзҡ„еӨ„зҗҶгҖӮ

йҖҡиҝҮдҝ®ж”№FuncnameGetCandidatesеҮҪж•°пјҢеңЁеҮҪж•°жҹҘжүҫж—¶еҢ№й…ҚеҢ…жӢ¬outеҸӮж•°еңЁеҶ…зҡ„жүҖжңүеҸӮж•°гҖӮ

еҮҪж•°зј–иҜ‘ж—¶пјҢжһ„йҖ дёҖдёӘrowеҸҳйҮҸжқҘе®№зәіOUTеҸӮж•°еҸҳйҮҸе’Ңиҝ”еӣһеҖјеҸҳйҮҸгҖӮдҝ®ж”№зј–иҜ‘еҘҪзҡ„еҮҪж•°иҝ”еӣһзұ»еһӢпјҲfunction→fn_rettypeпјүдёәRECORDOIDгҖӮ

еҮҪж•°жү§иЎҢж—¶пјҢеңЁExecInitFuncеҮҪж•°дёӯи°ғз”Ёж–°еҮҪж•°ExecInitFuncOutParamsпјҢ жһ„йҖ OUTеҸӮж•°и®Ўз®—иҠӮзӮ№пјҢйҖҡиҝҮи®Ўз®—еҮҪж•°plisql_out_paramе°ҶеҮҪж•°иҝ”еӣһеҖје’ҢOUTеҸӮж•°еҖјд»ҺtupleдёӯеҲҶзҰ»пјҢе№¶дё”дёәOUTеҸӮж•°еҗ‘еӨ–иөӢеҖјгҖӮ

1.2. еҢҝеҗҚеқ—ж”ҜжҢҒoutеҸӮж•°

дёәдәҶж”ҜжҢҒеҶ’еҸ·еҚ дҪҚз¬ҰеҪўејҸзҡ„з»‘е®ҡеҸҳйҮҸпјҢдҝ®ж”№ora_scan.lж–Үд»¶пјҢж·»еҠ иҜӯжі•пјҢйҒҮеҲ°еҶ’еҸ·еҚ дҪҚз¬Ұиҝ”еӣһORAPARAMгҖӮеңЁora_gram.yж–Үд»¶дёӯпјҢеңЁc_exprе’ҢиөӢеҖјplassign_targetзҡ„иҜӯжі•дёӯпјҢж·»еҠ еҜ№ORAPARAMж·»еҠ еӨ„зҗҶпјҢжһ„йҖ дёҖдёӘOraParamRefиҠӮзӮ№гҖӮ

ж–°еўһ DO + USING иҜӯжі•пјҡ

	DO [ LANGUAGE lang_name ] code [USING IN | OUT | IN OUT, ...]

дҝ®ж”№ora_gram.yж–Үд»¶пјҢеўһеҠ DO+USINGиҜӯжі•ж”ҜжҢҒгҖӮDoStmtз»“жһ„дҪ“дёӯеўһеҠ  paramsmode з”ЁдәҺеӯҳеӮЁеҢҝеҗҚеқ—дёӯзҡ„з»‘е®ҡеҸҳйҮҸжЁЎејҸзӯүдҝЎжҒҜзҡ„еҲ—иЎЁгҖӮ

еҜ№PBEиҝҮзЁӢзҡ„дҝ®ж”№еҢ…жӢ¬пјҡexec_parse_messageеҮҪж•°дёӯпјҢж №жҚ®еә”з”ЁжҺҘеҸЈдј иҝҮжқҘзҡ„еҸӮж•°зұ»еһӢoidпјҢиҜҶеҲ«еҮәе®һеҸӮзҡ„жЁЎејҸпјӣеңЁexec_bind_messageеҮҪж•°дёӯпјҢеҜ№дәҺеҢҝеҗҚеқ—DO+USINGпјҢиҜҶеҲ«еҮәUSINGеҗҺйқўеҸӮж•°зҡ„жЁЎејҸпјҢеҗ‘жү§иЎҢеҷЁдј йҖ’еҸӮж•°дҝЎжҒҜгҖӮ

еҗ«OUTеҸӮж•°еҢҝеҗҚеқ—зҡ„жү§иЎҢ

  1. еңЁPortalStartеҮҪж•°дёӯеҜ№еҢҝеҗҚеқ—иҜӯеҸҘпјҢи°ғз”ЁCreateTupleDescFromParamsеҮҪж•°пјҢеўһеҠ еҸӮж•°жһ„йҖ жҸҸиҝ°дҝЎжҒҜгҖӮ

  2. PLiSQL_functionжҲҗе‘ҳfn_prokindеўһеҠ ж–°еҖјPROKIND_ANONYMOUS_BLOCKз”ЁжқҘиЎЁзӨәеҢҝеҗҚеқ—гҖӮ

  3. еңЁplisql_exec_functionеҮҪж•°дёӯпјҢеҜ№жңүoutеҸӮж•°зҡ„еҢҝеҗҚеқ—еҒҡеҲӨж–ӯпјҢи°ғз”Ёplisql_anonymous_return_out_parameterеҮҪж•°е®ҢжҲҗеҜ№OUTеҸӮж•°жһ„йҖ PLiSQL_rowзұ»еһӢзҡ„еҸҳйҮҸпјҢжңҖеҗҺе°Ҷrowзұ»еһӢзҡ„еҸҳйҮҸжұӮеҖјеҪ“дҪңеҢҝеҗҚеқ—еҮҪж•°зҡ„иҝ”еӣһеҖјгҖӮ

1.3. libpqдёӯи°ғз”Ёеҗ«outеҸӮж•°зҡ„еҮҪж•°

дҝ®ж”№Libpqд»Ҙж”ҜжҢҒжҢүдҪҚзҪ®е’ҢжҢүеҸӮж•°еҗҚеӯ—з»‘е®ҡпјҢж¶үеҸҠSQLз«ҜпјҢPLiSQLз«ҜпјҢlibpqжҺҘеҸЈз«ҜгҖӮ

  1. SQLз«ҜпјҡеңЁжңҚеҠЎеҷЁз«Ҝе®һзҺ°зі»з»ҹеҮҪж•° get_parameter_descriptionпјҢиҜҘеҮҪж•°ж №жҚ®SQLиҜӯеҸҘпјҢиҝ”еӣһеҸҳйҮҸеҗҚеӯ—дёҺдҪҚзҪ®зҡ„е…ізі»гҖӮиҝҷдёӘеҮҪж•°иў«з”ЁеңЁlibpqжҺҘеҸЈеҮҪж•°дёӯгҖӮ иҝ”еӣһзҡ„з¬¬дёҖиЎҢnameжҳҫзӨәSQLзұ»еһӢпјҢеҗҺйқўиЎҢдҫқж¬ЎжҳҫзӨәеҚ дҪҚз¬ҰеҗҚеӯ—гҖҒдҪҚзҪ®дҝЎжҒҜгҖӮ

    ```
ivorysql=# select * from get_parameter_description('insert into t values(:x, :y);');
 name  | position
-------+----------
 false |        0
 :x    |        1
 :y    |        2
(3 rows)
```
еҜ№дәҺеҢҝеҗҚеқ—иҜӯеҸҘпјҢе°ҡдёҚж”ҜжҢҒгҖӮ

  2. PLiSQLз«Ҝпјҡдё»иҰҒжҳҜPL/iSQLеқ—ж №жҚ®еҸӮж•°дҪҚзҪ®жҲ–еҸӮж•°еҗҚз§°и°ғж•ҙеҸӮж•°еҶ…йғЁж ҮиҜҶгҖӮ

    жү§иЎҢеҮҪж•°йңҖиҰҒд»Һз»‘е®ҡеҸҘжҹ„дёӯиҺ·еҸ–еҸӮж•°зҡ„еҖјдёҺзұ»еһӢзҡ„дҝЎжҒҜпјӣ
    еҜ№outеҸӮж•°зҡ„иҝ”еӣһеҲ—еҗҚз§°еҒҡдёҖдёӘзү№ж®ҠеӨ„зҗҶгҖӮеҰӮжһңжҳҜoutеҸӮж•°пјҢйӮЈд№Ҳе…¶еҲ—еҗҚз§°дёә_column_xxxпјҢе…¶дёӯxxxжҳҜoutеҸӮж•°зҡ„дҪҚзҪ®пјҢд»ҺиҖҢж №жҚ®з»‘е®ҡдҪҚзҪ®дёҺиҝ”еӣһзҡ„дҪҚзҪ®д»Һз»“жһңйӣҶдёӯз»ҷoutеҸӮж•°иөӢеҖјпјӣ
    еңЁPLiSQLжү§иЎҢз«ҜпјҢж №жҚ®еҸӮж•°еҗҚеӯ—еҮәзҺ°зҡ„дҪҚзҪ®пјҢи°ғж•ҙеҗҚеӯ—иҪ¬жҚўжҲҗеҶ…йғЁж ҮиҜҶзҡ„еҸҳйҮҸпјҢеҰӮ$numberгҖӮеңЁиҝ”еӣһеҲ°е®ўжҲ·з«Ҝзҡ„ж—¶еҖҷпјҢеҸ‘йҖҒжҸҸиҝ°дҝЎжҒҜз»ҷlibpqпјҢ д»ҺиҖҢиҫҫеҲ°иҝ”еӣһзҡ„еҲ—еҗҚжҳҜз”ұеҸӮж•°еҗҚеӯ—жһ„йҖ пјҢLIBPQз«Ҝж №жҚ®еҲ—еҗҚжқҘз»ҷoutеҸӮж•°иөӢеҖјгҖӮ

  3. libpqжҺҘеҸЈз«ҜпјҡжҸҗдҫӣеҮҶеӨҮгҖҒз»‘е®ҡгҖҒжү§иЎҢеҮҪж•°пјҢиҝҷдәӣеҮҪж•°дёҺOCIжҺҘеҸЈзӣёеә”еҮҪж•°зұ»дјјгҖӮ
