outеҸӮж•°

1. зӣ®зҡ„

IvorySQLжҸҗдҫӣдәҶе…је®№Oracleзҡ„outеҸӮж•°еҠҹиғҪпјҢеҢ…жӢ¬еҗ«жңүoutеҸӮж•°зҡ„еҮҪж•°дёҺиҝҮзЁӢгҖҒеҢҝеҗҚеқ—ж”ҜжҢҒoutеҸӮж•°гҖҒlibpqж”ҜжҢҒoutеҸӮж•°гҖӮ

жң¬ж–ҮжЎЈж—ЁеңЁдёәдҪҝз”Ёдәәе‘ҳд»Ӣз»ҚoutеҸӮж•°зҡ„еҠҹиғҪгҖӮ

2. еҠҹиғҪиҜҙжҳҺ

IvorySQLжҸҗдҫӣдәҶе…је®№Oracleзҡ„outеҸӮж•°еҠҹиғҪпјҢеҢ…жӢ¬еҰӮдёӢеҶ…е®№гҖӮ

2.1. еҗ«жңүoutеҸӮж•°зҡ„еҮҪж•°

еҲӣе»әиҜӯжі•еҰӮдёӢпјҡ

CREATE [ OR REPLACE ] FUNCTION
    name ( [ [ argmode ] [ argname ] argtype [ { DEFAULT | = } default_expr ] [, ...] ] )
    [ RETURNS rettype
      | RETURNS TABLE ( column_name column_type [, ...] ) ]
  { LANGUAGE lang_name
    | TRANSFORM { FOR TYPE type_name } [, ... ]
    | WINDOW
    | { IMMUTABLE | STABLE | VOLATILE }
    | [ NOT ] LEAKPROOF
    | { CALLED ON NULL INPUT | RETURNS NULL ON NULL INPUT | STRICT }
    | { [ EXTERNAL ] SECURITY INVOKER | [ EXTERNAL ] SECURITY DEFINER }
    | PARALLEL { UNSAFE | RESTRICTED | SAFE }
    | COST execution_cost
    | ROWS result_rows
    | SUPPORT support_function
    | SET configuration_parameter { TO value | = value | FROM CURRENT }
    | AS 'definition'
    | AS 'obj_file', 'link_symbol'

  } ...

е…¶дёӯargmodeеҸҜд»ҘжҳҜINпјҢOUTпјҢINOUTпјҲIN OUTпјүпјҢVARIADICгҖӮеҰӮжһңдёҚжҢҮе®ҡй»ҳи®ӨжҳҜINгҖӮargmodeд№ҹеҸҜд»ҘеҶҷеңЁargnameзҡ„еҗҺйқўгҖӮ

дёҚеҗҢдәҺеҺҹз”ҹPGпјҢе…је®№Oracleзҡ„outеҸӮж•°е’Ңиҝ”еӣһеҖјж•°жҚ®зұ»еһӢжІЎжңүе…іиҒ”гҖӮIN OUTжЁЎејҸе’ҢOUTжЁЎејҸйғҪдёҚиғҪжңүй»ҳи®ӨеҖјгҖӮеҰӮжһңжңүoutеҸӮж•°пјҢеҲҷеҮҪж•°дҪ“дёӯеҝ…йЎ»жңүRETURNиҜӯеҸҘгҖӮ

2.2. еҢҝеҗҚеқ—ж”ҜжҢҒoutеҸӮж•°

ж”ҜжҢҒеҶ’еҸ·еҚ дҪҚз¬ҰеҪўејҸзҡ„з»‘е®ҡеҸҳйҮҸпјҢдҫӢеҰӮ: 1пјҢ:nameгҖӮ

ж–°еўһDO+USINGиҜӯжі•пјҡ DO [ LANGUAGE lang_name ] code [USING IN | OUT | IN OUT, …​]

ж”ҜжҢҒеңЁlibpqдёӯжҢүдҪҚзҪ®е’ҢжҢүеҸӮж•°еҗҚеӯ—з»‘е®ҡеҸҳйҮҸпјҢжҸҗдҫӣзі»з»ҹеҮҪж•°get_parameter_descrпјҢиҜҘеҮҪж•°ж №жҚ®SQLиҜӯеҸҘпјҢиҝ”еӣһеҸҳйҮҸеҗҚеӯ—дёҺдҪҚзҪ®зҡ„е…ізі»гҖӮ

2.3. libpqдёӯи°ғз”Ёеҗ«outеҸӮж•°зҡ„еҮҪж•°

libpqжҺҘеҸЈз«ҜжҸҗдҫӣеҮҶеӨҮгҖҒз»‘е®ҡгҖҒжү§иЎҢеҮҪж•°пјҢиҝҷдәӣеҮҪж•°дёҺOCIзӣёеә”еҮҪж•°зұ»дјјгҖӮ

дҪҝз”ЁжөҒзЁӢдёәпјҡ дҪҝз”ЁIvyHandleAllocеҲҶй…ҚиҜӯеҸҘеҸҘжҹ„е’Ңй”ҷиҜҜеҸҘжҹ„гҖӮ и°ғз”ЁIvyStmtPrepareеҮҶеӨҮиҜӯеҸҘгҖӮ и°ғз”ЁIvyBindByPosжҲ–IvyBindByName з»‘е®ҡеҸӮж•°гҖӮ и°ғз”ЁIvyStmtExecute жү§иЎҢпјҢеҸҜйҮҚеӨҚжү§иЎҢгҖӮ и°ғз”ЁIvyFreeHandle йҮҠж”ҫиҜӯеҸҘеҸҘжҹ„е’Ңй”ҷиҜҜеҸҘжҹ„гҖӮ

еҸҰеӨ–иҝҳе®һзҺ°дәҶIvyconnectdbпјҢIvystatusпјҢIvyexecпјҢIvyresultStatusпјҢIvyCreatePreparedStatementпјҢIvybindOutParameterByPosпјҢIvyexecPreparedStatementпјҢIvyexecPreparedStatement2пјҢIvynfieldsпјҢIvyntuplesпјҢIvyclearзӯүдёҖзі»еҲ—жҺҘеҸЈеҮҪж•°гҖӮ
