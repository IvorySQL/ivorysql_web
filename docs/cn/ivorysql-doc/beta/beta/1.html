1. 版本介绍

IvorySQL3.0 基于 PostgreSQL16.0 ，包含来自 PostgreSQL 16.0 的各种修复。有关 PostgreSQL 16.0中更详细的更新和错误修复，请参阅官方 PostgreSQL 16.0 发行说明 。
